## 快速测试步骤

### 1. 专家登录测试
```

```

### 2. 访问权限测试
- [ ] 登录成功后能看到仪表盘

- [ ] 点击"培训计划"不会提示权限错误

### 4. 培训详情测试
- [ ] 点击某个培训的"详情"按钮
- [ ] 详情模态框能正常打开
- [ ] 能看到培训基本信息

- [ ] 看不到客户电话、邮箱等敏感信息
